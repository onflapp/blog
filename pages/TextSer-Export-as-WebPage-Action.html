Export Markdown As Web Page

TextSer can generate HTML, CSS and images from any RTF, RTFD, markdown text files and DNoteIt document. This action can be combined together with other actions to from complete publishing workflow.

Destination name and Style name must correspond to the configuration in your TextSer preferences. The style refers to CSS style, whereas destination is action destination.

Use Advanced section to insert custom header or body content. This is very useful for inserting Google Tag Manager scripts or custom javascript. Make sure the content is valid HTML markup.
