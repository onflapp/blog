Cuts one PDF into multiple tiles. This will enable you to print one large document on multiple pages.

The destination paper size will set size of each ‘tile’. Specifying destination size larger then size of the source will merely reproduce the content.

You can cut the margins by specifying ‘tile overlap’

will produce 8 smaller pages
