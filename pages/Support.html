Using my apps

Although I tried my best to make my apps as easy to use as possible, they tend to be “technical” in nature. In order to fulfil very strict app store policy, some of the functionality is not as user-friendly (e.g. plugin system or reliance on helper apps). If you are confused, please let me know. I’ll try to help.

Reporting bugs

Furthermore, no app is perfect (unfortunately). If you ram bugs or crashes, please report them so that I can fix it in the next release.
