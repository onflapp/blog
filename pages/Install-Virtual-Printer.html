Virtual printer will help you to produce PDF from any application that can print. The VipRiser has been around since 2014 and is now part of VipRiser PDF Ops.

Installing and configuring the virtual PDF printer

The virtual printer has two parts, printer driver and user application which is used to congigure and control it.

  1. download the latest version from github page

  2. launch the VipRiser.app for the first time NOTE: you may need to unzip it first

  3. the app will lead you though the installation process

A the end you should end up with Print to VipRiser showing in your print dialog box.
