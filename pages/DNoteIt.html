DNoteIt

DNoteIt is innovative Markdown text editor. It works in hybrid mode, in which all links and images are directly embedded in the markup. This gives you nice WYSIWYG-like editing experience while writing in Markdown.

DNoteIt is available for Mac, iPhone, iPad and is even able to capture notes from your Apple Watch.

Hybrid editing mode

DNoteIt offers powerful syntax highlighting, with support for variable width and size fonts, margins and inline preview for embedded images and links. This can make the source markup look very similar to the final output.

Built-in Web preview will show the final output by transforming the markup to HTML without leaving the app. Both syntax highlighting and preview can be styled using CSS.

Document-based

Each page is stored as a separate document. A single document file contains text markup, metadata and any embedded image. This makes this fully self-contained format that can be easy shared or synchronized by any file-base mechanism that your Mac or iPhone/iPad supports (e.g. iCloud, DropBox, SharePoint, etc.).

DNoteIt documents are automatically versioned, made searchable via Spotlight full-text search and can be easily organized in folders or using tags.

Multi-device but Native

App’s user interface has been specifically created and tweaked for best usability to fit each type of a device. Therefore, DNoteIt running on a Mac will work like proper desktop app, whereas DNoteIt on iPhone will feel like an mobile app.

DNoteIt tries to leverage any native functionality and service offered by respective OS platform, such as scriptability, share extensions, previews and spell checking.
