CUPS-PDF Driver

Many CUPS-PDF users are switching to VipRiser because CUPS-PDFâ€™s lack of support for OS X 10.10. Each of the drivers, although having the same aim, takes a slightly different approach when it comes to PDF generation.

VipRiser generates PDFs directly, where as CUPS-PDF generates them from an intermediate PS (postscript file). Directly generated files are generally smaller and preserve web links. However, some users reported problems while post processing these files using pdftk and other such tools, therefore VipRiser includes an alternative driver that mimics the way CUPS-PDF works.

Installing VipRiser CUPS-PDF driver

  • got to System Preferences -> Printer & Scanners
  • add new printer (it may take few seconds for all drivers to appear)
  • select Print to VipRiser (CUPS-PDF)
