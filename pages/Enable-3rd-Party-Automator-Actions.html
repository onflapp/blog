Apple’s Mojave tighten up security restrictions for 3rd party automator actions. This means you will not be able to use VipRiser’s actions until you explicitly give permission to Automator to use these actions in your workflow. I think this is good thing overall, although a bit annoying. Fortunately, it needs to be enable just once.

To enable VipRiser’s workflow actions do the following:

1. Launch Automator

2. Go to “Third-party Automator Actions” setting

3. Enable the 3rd party actions

You are done
