VipRiser PDF Ops' Transform Action relies on local installation of Ghostscript processor. The Ghostscript is well known engine for working with PS and PDF files.

Easy Installation

  1. download pre-build package Ghostscript-9.52-Full.pkg
  2. install the package locally
  3. Use Transform Action to configure parameters

Advanced/Custom Installation

You can use any custom versions of the Ghostscript, if you wish to. The important thing is that the binary it is located at one of these locations:

/usr/local/bin/gs   
/opt/local/bin/gs   
/opt/bin/gs
