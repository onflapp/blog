VipRiser release 3.0 (commercial)

Complete Rewrite with new UI and features

NOTE: this version requires VipRiser PDF Ops to be installed (paid) to unlock advanced features

  • supports Mac OS 10.14
  • filters to transform PDF in various ways (CMYK, grayscale, compress images, PDF-X)
  • fully customizable workflows
  • built-in support for publishing to Kindle and iBooks
  • can act as default PDF handler to let you choose action you want to take interactively
  • support for Mac OS’s user switching (multiple instances of VipRiser can run concurrently)
  • new PPDs for Kindle and Screen output
